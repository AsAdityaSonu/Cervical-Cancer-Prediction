# Cervical-Cancer-Prediction

This repository contains a Python module for predicting cervical cancer using machine learning (ML) models, specifically leveraging the power of transformer architectures. The goal of this project is to provide a reliable tool for early detection of cervical cancer, aiding healthcare professionals in diagnosis and treatment planning.

## Features

- **Preprocessing**: Data cleaning, normalization, and feature extraction tailored for medical datasets.
- **Transformer Models**: Implementation of state-of-the-art transformer models for high accuracy in predictions.
- **Training Pipeline**: End-to-end training pipeline including data splitting, model training, validation, and hyperparameter tuning.
- **Evaluation Metrics**: Comprehensive set of evaluation metrics such as accuracy, precision, recall, F1-score, ROC-AUC, and confusion matrix.
- **Model Saving and Loading**: Easy model saving and loading functionalities for future predictions and analysis.
- **Visualization**: Tools for visualizing model performance and feature importance.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/asadityasonu/cervical-cancer-prediction.git
    ```
2. Navigate to the project directory:
    ```bash
    cd cervical-cancer-prediction
    ```
3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

### Data Preparation

1. Download the datasets from the following links:
    - [Cervical Cancer Largest Dataset (SipakMeD) on Kaggle](https://www.kaggle.com/datasets/prahladmehandiratta/cervical-cancer-largest-dataset-sipakmed)
    - [224x224 Cervical Cancer Screening on Kaggle](https://www.kaggle.com/datasets/ofriharel/224-224-cervical-cancer-screening)
    - [Cervical Cancer Screening Dataset on Mendeley](https://data.mendeley.com/datasets/zddtpgzv63/4)

2. Place your dataset in the `data/` directory.
3. Ensure the dataset follows the required format (details in the documentation).
